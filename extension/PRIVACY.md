# 隐私说明

`Personal Browser Home` 是个人新标签页扩展，默认设计目标是本地优先。

## 本地数据

以下数据保存在浏览器本地存储中：

- 快捷入口和搜索引擎配置
- 天气位置和模块开关
- 待办、笔记、应用内操作历史
- 设置页导入的备份数据

## 浏览器数据

扩展会在页面中读取并展示：

- 浏览器书签
- 浏览器历史
- 浏览器下载记录

这些数据仅用于当前新标签页界面展示，不会被上传、共享或同步到第三方服务。

## 外部请求

扩展会向 Open-Meteo 请求天气数据，向 Stooq 请求市场行情。请求中不包含浏览器书签、历史、下载记录、待办或笔记内容。

## 数据导出

设置页提供 JSON 备份和恢复能力。导出的文件由用户主动保存和管理，扩展不会自动上传备份文件。
